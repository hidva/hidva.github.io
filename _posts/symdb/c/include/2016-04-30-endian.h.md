*   字节序相关

## __BYTE_ORDER

*   用于在**预处理阶段**确定当前机器是大端模式,还是小端模式.可以如下使用:

    ```cpp
    #include <endian.h>
    #include <stdio.h>

    int
    main(int ,char **)
    {
    #if __BYTE_ORDER == __LITTLE_ENDIAN
        puts("小端模式");
    #elif __BYTE_ORDER == __BIG_ENDIAN
        puts("大端模式");
    #else
    #error 无法判断当前机器大小端模式
    #endif
        return 0;
    }
    ```

## 字节序转换

```c
#define htobe16(x) // 'htobe' -> host byte order to big endian;
#define htole16(x) // 'htole' -> host byte order to little endian;
#define be16toh(x) // 'betoh' -> big endian to host;
#define le16toh(x) // 'letoh' -> little endian to host;

#define htobe32(x)
#define htole32(x)
#define be32toh(x)
#define le32toh(x)

#define htobe64(x)
#define htole64(x)
#define be64toh(x)
#define le64toh(x)
```
